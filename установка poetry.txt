Если вы читаете этот комментарий, значит вы уже давно скачали poetry   с официального сайта, но как продвинуться дальше ? Вот алгоритм

Шаг номер ноль : Обзаводимся PyCharm Pro. Покупаем на Авито за 600 р лицензию на год.

1. Создаём папку будущего проекта в powershell или в gitbush пишем mkdir name_my_project

2. В PyCharm идём в настройки и отключаем интерпретатор. No intetpretator

3. Создаём виртуальное окружение в терминале

python -m venv venv

 

4. Активируем виртуальное окружение

./venv/Scripts/activate

 

5. Добавляем poetry в venv

pip install poetry

 

6. Активируем poetry

poetry init

 

7. Дальше со всем соглашаемся — жмём Enter и yes

 

8. Устанавливаем Джанго

poetry add django

 

9. Создаём файлы проекта

django-admin startproject config ./

 

10. Вносим изменения в manage.py

python manage.py migrate

( в результате этого действия появляется файл с базой данных db.sqlite3)

 

11. Идём в правую сторону экрана. Кликаем по иконке «База Данных». Нажимаем в выпавшем окне «+». Выбираем верхнюю позицию data source. Слева всплывает окно с разными базами данных. Выбираем SQLite3. Выпадает окно Data sources and drivers. Вкладка General, верхняя строка File. Прописываем путь до директории, которую мы создали в шаге 1. Нажимаем внизу окна Test Connection. Он должен быть Succeeded. Нажимаем Оk. Нажимаем обновить в планке, где нажимали на «+». В файле main появляются 12 таблиц.

 

12. Создаём первого юзера-администратора

python manage.py createsuperuser

придумываем имя, почту, пароль ( пароль не выводится в терминал, не забывайте)

 

13. Создаём приложение

python manage.py startapp user

 

14. Запускаем сервер

python manage.py runserver
